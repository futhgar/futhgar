I mostly work on three things: **AI agents** that automate the boring parts of running operations — monitoring, alerting, document processing, all running with safety rails. **Predictive maintenance** systems that tell you what's going to break before it does — vibration analysis, sensor pipelines, the whole stack from sensor to dashboard. And **infrastructure** on bare metal — Kubernetes clusters, GitOps, GPU compute, the kind of setup where "just use AWS" isn't an option.

`Python` `TypeScript` `Go` `Kubernetes` `ArgoCD` `OpenTofu` `Proxmox` `Prometheus` `Grafana` `PostgreSQL` `Qdrant`
